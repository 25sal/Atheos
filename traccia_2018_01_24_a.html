Traccia di Elementi di Programmazione



Sviluppare in linguaggio C un programma che:


  • legge una stringa da un file di testo “input.txt”.

  • salva in un secondo vettore di caratteri le lettere presenti nella stringa prese una sola voltaa

  • per ogni lettera del secondo vettore crea una struttura i cui campi sono:la lettera, la sua frequenza di occorrenza nella stringa, il vettore delle posizioni, il numero di occorrenze

    struct Mystruct

{

char lettera;

float frequenzaOccorrenza;

int numeroOccorrenze;

int posizioniLettera[100];

};

  • il vettore salva I SOLI ELEMENTI del vettore delle strutture ordinato in senso decrescente del numero di occorrenze in un file binario “output.bin”;



Esempio:


Stringa iniziale: “casa sacca”

Vettore di strutture:

‘a’, 4, 0.4, [1,3,6,9]

‘c’, 3, 0.3,[0,7,8]

‘s’,2, 0.2, [2,5]

‘ ’,1, 0.1, [4]
